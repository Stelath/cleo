## Introduction

The VITURE Glasses SDK is a comprehensive software development kit designed to provide developers with the tools and APIs necessary to integrate and control VITURE XR glasses devices in their applications. This SDK enables seamless communication with VITURE glasses hardware, allowing for advanced XR experiences, device management, and real-time data streaming.

## Key Features

- **Device Lifecycle Management**: Create, initialize, start, stop, and destroy device provider instances
- **Real-time Data Streaming**: Access IMU data, pose information, and camera frames
- **Device Control**: Adjust display modes, brightness, volume, and other device settings
- **Cross-Platform Support**: Compatible with Linux, macOS, Windows and Android platforms
- **Callback System**: Register callbacks for state changes and data updates
- **Logging Integration**: Customizable logging hooks for debugging and monitoring

## Architecture

The SDK follows a provider-based architecture where:

- `XRDeviceProvider` serves as the central interface for device interaction
- Device-specific APIs are available for different VITURE models (Carina vs. Gen1/Gen2)
- Protocol functions handle low-level USB communication and device control
- Callback mechanisms enable asynchronous data reception and state monitoring

## Requirements

- C/C++ development environment
- Linux (x64 / arm64): GLIBC _2.34, GLIBCXX_ 3.4.30
- macOS (arm64): Sequoia
- Windows (x64): Windows 10 or later
- Android (arm64): Android 8 (Oreo) or later
- USB access to VITURE glasses device
- Linux: udev rules for device access
- Android: Runtime USB device permission
